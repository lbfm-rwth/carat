Program Same_generators

Usage

Same_generators 'file1' 'file2' ['file3' .....]
file1: bravais_TYP containing the point group G.
file2: bravais_TYP containing the space group R1.
filex: bravais_TYP containing additional space groups.

Description

For each group in the files 'file2', 'file3', ... the program tries to find elements in the space group with the same linear part as the generators in file1.

Remarks

WARNING: If this is impossible, the program will run into an infinite loop.

See also for Extensions/Vector_systems, Extract and Z_equiv.
