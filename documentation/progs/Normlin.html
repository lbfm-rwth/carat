Program Normlin

Usage

Normlin 'file1' 'file2'
file1: matrix_TYP.
file2: matrix_TYP.

Description

Calculates for each matrix A in 'file2' a matrix X with the property that

SUMj Xi,j Fj = Atr Fj A

with Fj in 'file1'.

Remarks

CAUTION: The matrix describes the action on rows!
