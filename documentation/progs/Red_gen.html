Program Red_gen

Usage

Red_gen 'file'
file: bravais_TYP

Description

Tries to reduce the number of generators for the group in file. It will output a bravais_TYP, where the generators are a subset of the given generator such that they generate the same group.

Remarks

NOTE: The group has to be finite!

See also for Is_finite and Order.
