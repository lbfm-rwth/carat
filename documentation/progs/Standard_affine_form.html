Program Standard_affine_form

Usage

Standard_affine_form 'file1' 'file2' [-t]
file1: bravais_TYP containing a space group G.
file2: matrix_TYP contatining a presentation of the point group of G.

Description

Transforms the space group with generators in file1 into standard form, ie. the translation lattice is transformed to Zn. In case the translation subgroup has dimension smaller than n, i. e. the group is not an n-dimensional space group, the program will indicate an error, gives the rank, and exit.

Options

-t    : The transforming matrix will be given as well.

Remarks

This program is a synonym for Extract -t. See also for Extensions and Vector_systems.

Examples

  1. Do given matrices generate a space group? If so, find its name and normal representation.
