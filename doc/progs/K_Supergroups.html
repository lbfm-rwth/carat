Program KSupergroups

Usage

KSupergroups  'file1' [-h] [-a [-n]] [-f] [-t] [-p 'file2'] 'p_1' ... 'p_n'
file1: Spacegroup in standard affine form (without translations)
file2: (optional) Pointgroup P of R (with correct normalizer)
pi : primes

Description

Calculates the minimal klassengleich supergroups with pi-power index.

Options

-h   : Give a short help.
-a   : Calculate supergroups up to conjugation under
       the affine normalizer of R.
-n   : Print the lengthes of the orbits
       (this only works together with the option a).
-f   : Print the supergroups in files 'file1_j'.
-t   : Omit the last dim R generators. 
-d   : Only for debugging!

Remarks

See also for Graph, KSubgroups and Standard_affine_form.

Examples

  1. Calculate minimal/maximal klassengleich/translationengleich super-/sub-groups of a given space group up to conjugation under the affine normalizer of this group.
