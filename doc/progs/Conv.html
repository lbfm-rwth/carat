Program Conv

Usage

Conv 'file' ['name'] -g -G -m -M -t -T
file: matrix_TYP or bravais_TYP
name: (optional) a word

Description

Converts the matrix_TYP or the generators of the bravais_TYP into the format of other programs available. The options specify the program, and only one of them is allowed at one time. First of all the number of matrices is printed. The matrices are called 'name.1', 'name.2', ... or if name isn't given 'a.1', 'a.2', ...

Options

-g or -G: GAP-format
-m or -M: MAPLE-format
-t or -T: TeX-format (the matrices aren't named and
          the number of matrices isn't given)
