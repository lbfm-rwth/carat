Program First_perfect

Usage

First_perfect 'file1' 'file2' 'file3'
file1: bravais_TYP containing the finite unimodular group G.
file2: bravais_TYP containing Gtr.
file3: matrix_TYP containing a positive definite G-invariant form F.

Description

Returns a G-perfect form in the neighbourhood of F.

Remarks

See also for Normalizer and Perfect_neighbours.
