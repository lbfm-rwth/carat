Program Bravais_equiv

Usage

Bravais_equiv file1 file2
file1: bravais_TYP containing G.
file2: bravais_TYP containing H.

Description

Tests whether the BRAVAIS GROUPS of the groups G and H respectively are conjugated in GLn(Z). If so, it returns a conjugating matrix X which conjugates the BRAVAIS GROUPS, ie. X-1 B(G) X = B(H).

Remarks

WARNING: The procedure may involve calculating the normalizer of the groups. This may be very time comsuming, especially when both groups are <-In>, where n > 5.
