Program P_lse_solve

Usage

P_lse_solve 'file1' 'file2' -p=prime
file1: matrix A
file2: matrix B

Description

If 'file2' is not given, B = 0.
The program solves the linear equations 
 A * X = B
modulo the given prime (default 1949).
