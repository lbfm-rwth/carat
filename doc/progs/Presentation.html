Program Presentation

Usage

Presentation 'file' [-D]
file: bravais_TYP containing a finite matrix group G.

Description

Calculates a presentation of the finite group G.

Options

-D  : meant for debugging. Do not use.

Remarks

See also for Is_finite.

Examples

  1. Do given matrices generate a space group? If so, find its name and normal representation.
  2. Find all Z-classes, affine classes and torsion free space groups in a given Q-class.
