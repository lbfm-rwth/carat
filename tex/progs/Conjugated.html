Program Conjugated

Usage

Conjugated file1 file2 file3
file1: bravais_TYP containing the finite group G.
file2: bravais_TYP containing the finite group H.
file2: bravais_TYP containing the group N.

Description

The program checks whether there is an x in N with the property that x H x-1 <= G, and returns this x if it exists.

Remarks

WARNING: The program assumes that the orbit of H under N is finite.
