Program Normalizer_in_N

Usage

Normalizer_in_N 'file1' 'file2' [-f]
file1: bravais_TYP
file2: bravais_TYP

Description

Calculates the normalizer of the group in file1 in the group generated by the matrices of file2->gen, file2->cen and file2->normal.

Options

-f     :  The program assumes that the calculated normalizer
          is finite, and performs an base/strong generator
          algorithm on it. In effect you will get less
          generators for it. IT WILL WORK IF AND ONLY IF
          THE CALCULATED NORMALIZER IS INDEED FINITE.

Remarks

NOTE: The programm will work for option -f if and only if the calculated normalizer is indeed finite.
