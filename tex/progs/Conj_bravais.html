Program Conj_bravais

Usage

Conj_bravais 'file1' 'file2' [-i]
file1: bravais_TYP containing the group G.
file2: matrix_TYP containing a single matrix T.

Description

The program calculates TGT-1 and transforms all relevant data of G. The result is a bravais_TYP, that is written to standart output.

Options

 -i     : Transform the group by T^{-1} G T.

Examples

  1. Find all space groups with a given point group and decide for which superlattices each extension splits.
